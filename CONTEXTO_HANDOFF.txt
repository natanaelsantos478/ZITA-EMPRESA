1. O QUE É O SOFTWARE
================================================================================

ZITA é uma plataforma SaaS de gerenciamento de equipes de IAs para empresas.
Stack: React 19 + TypeScript + Vite + TailwindCSS + Supabase.

Funcionalidades principais:
- Organograma de IAs com 5 visualizações: Canvas, Retrô, Moderno, Profissional, 3D
- Escritório 3D interativo com avatares animados (Three.js), FPS controls, speech bubbles
- Ecossistema de IAs: memória compartilhada, fila de ações assíncronas, hierarquia
- Chat com cada IA individualmente
- Agente Zeus: IA mestre que dá ordens e recebe relatórios de todas as outras
- Integração com Gemini (Google AI Studio) via API key por agente

Tabelas Supabase relevantes:
  ia_agents    (25 cols) — cada IA registrada com hierarquia, capacidades, personalidade
  ia_conversas (15 cols) — histórico de conversas
  ia_mensagens (17 cols) — mensagens individuais
  ia_tarefas   (22 cols) — fila de tarefas
  ia_memorias  (13 cols) — memória compartilhada privada/equipe/global
  ia_acoes     (17 cols) — fila de ações assíncronas agente↔agente

6. COMO O ECOSSISTEMA FUNCIONA (RESUMO TÉCNICO)
================================================================================

FLUXO INBOUND (receber evento externo → agente processa):
  Sistema externo → POST /api-webhook?agent_id=X&company_id=Y
  → insere ia_acoes { status: 'pendente' }
  → browser (useEcosystem hook) detecta via realtime ou polling 30s
  → EcosystemEngine.processAction() roda:
     1. Claim otimista (UPDATE WHERE status='pendente')
     2. _callGemini() → Gemini retorna JSON com resposta + api_calls
     3. _executeTools() → executa cada api_call (WhatsApp/Slack/Notion/webhook)
     4. Salva resultado em ia_acoes { status: 'concluida' }
     5. Se pergunta/comando → envia relatorio de volta ao remetente
     6. Se delegar_para → cria novas ia_acoes para outros agentes

FLUXO OUTBOUND (agente decide chamar API externa):
  Gemini retorna:
  {
    "resposta": "Email enviado!",
    "api_calls": [
      { "integration": "slack", "action": "send_message",
        "params": { "channel": "#vendas", "message": "Novo lead captado!" } }
    ]
  }
  → EcosystemEngine._executeTools() executa o POST para Slack API
  → resultado salvo em ia_acoes.resultado.tool_results

MEMÓRIA COMPARTILHADA:
  ia_memorias: privada (só o agente), equipe (todos da empresa), global (idem)
  Injetada no system_prompt antes de cada chamada ao Gemini
  Agente pode pedir para salvar via "salvar_como_memoria": true na resposta

ZEUS 24/7:
  pg_cron a cada 2 min → chama zeus-scheduler Edge Function → processa fila do Zeus
  Funciona mesmo com todos os browsers fechados

--- supabase/functions/api-webhook/index.ts (CRIADO, precisa deploy) ---

import { serve }        from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'

const CORS = {
  'Access-Control-Allow-Origin':  '*',
  'Access-Control-Allow-Methods': 'POST, OPTIONS',
  'Access-Control-Allow-Headers': 'Content-Type, Authorization, x-agent-id, x-company-id, x-source, x-priority',
}

serve(async (req: Request) => {
  if (req.method === 'OPTIONS') return new Response(null, { headers: CORS })
  if (req.method !== 'POST') {
    return new Response(JSON.stringify({ error: 'Método não permitido. Use POST.' }), {
      status: 405, headers: { ...CORS, 'Content-Type': 'application/json' },
    })
  }
  try {
    const url       = new URL(req.url)
    const agentId   = url.searchParams.get('agent_id')   ?? req.headers.get('x-agent-id')
    const companyId = url.searchParams.get('company_id') ?? req.headers.get('x-company-id')
    const source    = url.searchParams.get('source')     ?? req.headers.get('x-source') ?? 'webhook'
    const priority  = url.searchParams.get('priority')   ?? req.headers.get('x-priority') ?? 'normal'

    if (!agentId || !companyId) {
      return new Response(
        JSON.stringify({ error: 'agent_id e company_id são obrigatórios' }),
        { status: 400, headers: { ...CORS, 'Content-Type': 'application/json' } }
      )
    }

    let payload: Record<string, unknown> = {}
    const ct = req.headers.get('content-type') ?? ''
    if (ct.includes('application/json')) {
      payload = await req.json().catch(() => ({}))
    } else if (ct.includes('application/x-www-form-urlencoded')) {
      const text = await req.text()
      new URLSearchParams(text).forEach((v, k) => { payload[k] = v })
    }

    const supabase = createClient(
      Deno.env.get('SUPABASE_URL')!,
      Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!
    )

    const { data, error } = await supabase
      .from('ia_acoes')
      .insert({
        company_id:    companyId,
        de_agent_id:   null,
        para_agent_id: agentId,
        tipo:          'comando',
        prioridade:    ['baixa', 'normal', 'alta', 'urgente'].includes(priority) ? priority : 'normal',
        payload:       { source, ...payload, _webhook_received_at: new Date().toISOString() },
        executar_apos: new Date().toISOString(),
        expira_em:     new Date(Date.now() + 7 * 86_400_000).toISOString(),
      })
      .select('id')
      .single()

    if (error) throw new Error(error.message)

    return new Response(
      JSON.stringify({ ok: true, acao_id: data.id, agent_id: agentId, source }),
      { status: 200, headers: { ...CORS, 'Content-Type': 'application/json' } }
    )
  } catch (err) {
    const msg = err instanceof Error ? err.message : String(err)
    return new Response(
      JSON.stringify({ error: msg }),
      { status: 500, headers: { ...CORS, 'Content-Type': 'application/json' } }
    )
  }
})
